GUI Connect 4

For Python 3.7.2

To run the program make sure you are in the scripts directory and run the program using "py main.py"

Connect 4 is a 2 player game so you will need two players to play. 

Click over a row with black circles in it and try to win the game of connect 4 by getting a horizontal, vertical or diagonally adjacent match of 'Y",
or 'X' 4 times.
